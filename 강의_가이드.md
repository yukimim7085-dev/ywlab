# 주식 입문자를 위한 경제 기초 교육 - 강의 가이드

## 강의 개요
- **대상**: 주식/투자 입문자, 경제 기초 지식이 필요한 분
- **총 분량**: 26개 섹션
- **권장 강의 시간**: 6~8시간 (2~3회 분할 권장)

---

## 강의 구성안

### 📚 1일차: 경제 기초와 자산 이해 (2~2.5시간)

#### 모듈 1: 오프닝 & 경제 기초 (40분)
```
[도입 질문]
"여러분, 금리가 오르면 주식은 오를까요, 내릴까요?"
"연준(Fed)이 뭔지 아시는 분?"

[강의 내용]
1. 중앙은행과 연준 (15분)
   - 핵심: "금리를 누가 정하는가?"
   - 예시: FOMC 발표 때 주식시장 반응 보여주기

2. 단리와 복리 (10분)
   - 핵심: 72의 법칙
   - 실습: "10년 후 얼마가 될까?" 계산해보기

3. 채권의 기초 (15분)
   - 핵심: 금리↑ = 채권가격↓ (가장 중요!)
   - 비유: "시소처럼 반대로 움직인다"

[체크 질문]
"금리가 오르면 기존 채권의 매력은 올라갈까요, 내려갈까요?"
```

#### 모듈 2: 자산의 종류 (30분)
```
[도입 질문]
"불안할 때 사람들은 어디에 돈을 넣을까요?"

[강의 내용]
1. 안전자산 vs 위험자산 vs 대체자산 (15분)
   - 시각자료: 자산 분류표 보여주기
   - 핵심: "역의 상관관계"

2. 경제 상황별 자산 움직임 (15분)
   - 4계절 이론 설명
   - 예시: 2022년 금리 인상기 자산 변화

[참여 활동]
"지금 경제가 4계절 중 어디라고 생각하세요?"
```

#### 모듈 3: 경기사이클 (25분)
```
[도입]
"경제도 계절처럼 순환합니다"

[강의 내용]
1. 경기사이클 4단계 (15분)
   - 회복 → 확장 → 수축 → 침체
   - 각 단계별 투자 전략

2. 코스톨라니 달걀 이론 (10분)
   - 핵심: "남들이 두려워할 때 사라"
   - 실제 사례: 2020년 코로나 폭락 후 반등

[마무리 질문]
"경기 바닥에서 주식을 사면 늦을까요, 적절할까요?"
```

#### 모듈 4: 환율과 배당 (25분)
```
[강의 내용]
1. 환율과 투자 (15분)
   - 핵심: 환율↑ = 수출주 유리
   - 예시: 삼성전자, 현대차 환율 민감도

2. 배당 투자 기초 (10분)
   - 핵심: 배당락일 전에 사야 배당 받음
   - T+2 규칙 강조

[실습]
"삼성전자 배당수익률 계산해보기"
```

### 📚 2일차: 주식 시장과 분석 (2~2.5시간)

#### 모듈 5: 주식 시장 이해 (30분)
```
[도입 질문]
"코스피와 코스닥의 차이를 아시나요?"

[강의 내용]
1. 주요 증시 & 지수 (15분)
   - 한국: KOSPI, KOSDAQ
   - 미국: S&P500, 나스닥, 다우존스

2. 주식 용어 총정리 (15분)
   - 매수/매도, 시가/종가
   - 커뮤니티 속어: 익절, 손절, 존버

[퀴즈]
"상한가는 몇 %? 하한가는?"
```

#### 모듈 6: 기업 분석 지표 (40분)
```
[도입]
"좋은 주식을 어떻게 찾을까요?"

[강의 내용]
1. 시가총액 (10분)
   - 계산: 주가 × 발행주식수
   - 대형주/중형주/소형주 분류

2. PER / PBR / ROE / EPS (20분)
   - PER: "주가가 이익의 몇 배인가?"
   - 실습: 네이버 금융에서 직접 찾아보기

3. 재무제표 읽는 법 (10분)
   - 3대 재무제표 소개
   - DART 사용법 간단 시연

[실습]
"삼성전자 PER, PBR 직접 찾아보기"
```

#### 모듈 7: 기술적 분석 기초 (35분)
```
[도입]
"차트로 매매 타이밍을 잡을 수 있을까요?"

[강의 내용]
1. 이동평균선 (10분)
   - 5일, 20일, 60일, 120일선
   - 골든크로스 vs 데드크로스

2. RSI / MACD / 볼린저밴드 (15분)
   - RSI: 70 이상 과매수, 30 이하 과매도
   - 실제 차트로 예시 보여주기

3. 피보나치 되돌림 (10분)
   - 61.8% 황금비율
   - 지지/저항선 개념

[주의사항 강조]
"기술적 분석은 100% 맞지 않습니다. 참고용!"
```

#### 모듈 8: 실적과 공매도 (25분)
```
[강의 내용]
1. 실적 시즌과 어닝 (15분)
   - 컨센서스, 서프라이즈, 쇼크
   - 예시: 실적 발표 후 주가 변동 사례
```

### 📚 3일차: 실전 투자 전략 (2~2.5시간)

#### 모듈 9: 투자 상품 (35분)
```
[도입 질문]
"처음 투자한다면 개별주식 vs ETF?"

[강의 내용]
1. ETF 투자 (20분)
   - 장점: 분산투자, 낮은 수수료
   - 종류: 지수추종, 섹터, 채권, 금
   - 주의: 레버리지/인버스 ETF 위험성

2. IPO와 공모주 청약 (15분)
   - 청약 절차 5단계
   - 균등배정 vs 비례배정
   - 따상의 의미

[실습]
"현재 공모주 일정 확인해보기"
```

#### 모듈 10: 자산배분과 절세 (35분)
```
[강의 내용]
1. 올웨더 포트폴리오 (15분)
   - 주식 30%, 장기채 40%, 중기채 15%, 금 7.5%, 원자재 7.5%
   - 리밸런싱 개념

2. 섹터/산업 분류 (10분)
   - GICS 11대 섹터
   - 경기민감주 vs 방어주

3. 절세 계좌 총정리 (10분)
   - ISA, 연금저축, IRP 비교
   - 최적 납입 순서

[핵심 강조]
"세금도 수익률의 일부입니다!"
```

#### 모듈 11: 세금과 리스크 관리 (30분)
```
[강의 내용]
1. 주식 세금 기초 (10분)
   - 국내: 배당소득세 15.4%, 양도세 없음(일반인)
   - 해외: 250만원 초과 시 22%

2. 리스크 관리 (20분)
   - 손절매: 초보자는 -5~10%
   - 분산투자: 최소 5~10개 종목
   - 포지션 사이징: 종목당 최대 20%
   - 손실 회복 수익률 표 보여주기

[강조]
"50% 잃으면 100% 벌어야 본전입니다"
```

#### 모듈 12: 종합 정리 & Q&A (20분)
```
[최종 정리]
1. 핵심 10가지 복습
   - 금리↑ = 채권↓, 주식↓
   - 분산투자는 필수
   - 손절 라인 미리 정하기
   - 여유자금으로만 투자
   - 장기 복리의 힘

2. 초보자 체크리스트 복습

3. Q&A 시간

[마무리 멘트]
"투자는 마라톤입니다. 천천히, 꾸준히, 공부하면서!"
```

## 강의 팁

### 효과적인 전달법
1. **실제 사례 활용**: 삼성전자, 테슬라 등 익숙한 기업 예시
2. **실시간 데모**: 네이버 금융, DART 직접 보여주기
3. **참여 유도**: 매 섹션마다 질문 던지기
4. **비유 사용**: "시소", "계절", "성적표" 등 쉬운 비유

### 주의사항
1. **면책 고지**: "투자 권유가 아닌 교육 목적"
2. **과장 금지**: "무조건 오른다" 같은 표현 X
3. **리스크 강조**: 원금 손실 가능성 반복 언급

### 준비물
- 노트북/태블릿 (실시간 차트 시연용)
- 네이버 금융, DART 즐겨찾기
- 프레젠테이션 자료 (교육 HTML과 연동)
